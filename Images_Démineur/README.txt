Le dossier Images_Démineur contient les différents documents (images) nécessaires au fonctionnement du jeu.
Il est nécessaire de placer ce dossier sur l'ordinateur de la manière suivante :

C:\Program Files (x86)\Images_Démineur

Cela permet de lancer le jeu en laissant le raccourci sur le bureau

Il y a les photos utilisées par le démineur sous la forme Pic[n] (avec n allant de 0 à 8).
Elles ont une résolution de 60x60 et ne sont pas transparentes.

Le nombre de bombes est modifiable dans la variable "NBBOMBES"
La largeur du tableau  est modifiale dans la variable "LargeurGrille"
Cela peut-être utile si la grille ne rentre pas entièrement dans l'écran
