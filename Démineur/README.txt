La dernière version est la V1.7.0
Elle affiche la grille avec les images sur chaque case.
Il faut que le programme V1.7.0 soit dans le dossier avec les autres images pour qu'il affiche les images.

Il y a les photos orginales sous la forme Pic[n]BIG.png (avec n allant de 0 à 8).
Elles ont une résolution de 800x800 et sont transparentes.
Il y a les photos utilisées par le démineur sous la forme Pic[n] (avec n allant de 0 à 8).
Elles ont une résolution de 60x60 et ne sont pas transparentes.

Le nombre de bombes par défaut est fixé à 20, mais il est modifiable dans la variable "NBBOMBES"
